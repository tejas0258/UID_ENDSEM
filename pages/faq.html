Frequently Asked Questions

Quick answers for shopping, cart storage, signup, login, product details, and project testing.

What products are available on VELOCE Labs?

VELOCE Labs lists robotics and electronics components like microcontrollers, development boards, sensors, batteries, chargers, drivers, and project modules.

How do I add a product to the cart?

Open the shop page, choose a product, and press the Add to Cart button. The cart icon updates automatically and the cart page displays the selected items.

Will my cart stay after refreshing the page?

Yes. The cart uses browser storage with fallback support, so items should stay visible during normal project testing and page refreshes.

Is the signup page connected to a real database?

No. This is a frontend project demo. Signup details are saved in the browser using local storage so login, logout, and profile features can work without a backend.

How do login and signup work?

Create an account from the signup page, then login with the same email and password. After login, the profile page shows your saved user identity.

How can I view product details?

Click a product image or product name from the shop. The detail page shows the product overview, technical data, dimensions, and an Add to Cart button.

Which framework or libraries are used?

The project uses Tailwind CDN for visible UI blocks and AOS for scroll animations. Custom styling remains in css/style.css, and page logic remains in js/app.js.

Can this become a real ecommerce website?

Yes. The current version is frontend-only. A real version would need backend authentication, a product database, server-side cart storage, and payment integration.
