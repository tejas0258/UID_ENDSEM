# VELOCE LABS ⚡
A dark-themed electronics e-commerce site built with vanilla HTML, CSS, and JavaScript. No frameworks, no backend — just localStorage.

**[Live Demo](https://tejas0258.github.io/UID_ENDSEM/)**

## Pages
- Home, Products, Product Detail
- Cart → Checkout (UPI / Card / COD)
- Login, Sign Up, Profile, About, FAQ

## Running It
Just open `index.html` in a browser. Or use Live Server in VS Code.

## Adding Products
Drop your image in `assets/products/` and add an entry to the products array in `js/app.js`.

## Stack
